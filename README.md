# Portfolio

Personal portfolio site for **Eshika Singha** — a full-stack engineer at 24/7 Software.

A single-page React + Vite site that introduces who I am, the work I do, and the things I've shipped: backend services in Java/Spring Boot, React frontends, third-party integrations (Qrvey for embedded analytics & dashboards), and AI side projects like SketchMind and Election Assistant India.

The whole site is one component file, styled with plain CSS, no UI library — meant to be easy to scan and easy to tweak.

🌐 Live at [eshikasingha.netlify.app](https://eshikasingha.netlify.app)
